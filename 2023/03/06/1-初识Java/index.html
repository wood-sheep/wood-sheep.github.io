2.java历史

1998年，sun公司发布了第二代java平台，这个平台中包含了3个版本的 java
javaME
应用于移动端，无线及有限资源环境
用的很少，做机顶盒，光猫这种微型设备的一些开发，市场占有
几乎淘汰
javaSE
应用于桌面环境
java基础，面向对象，反射，多线程
javaEE
应用于java的应用服务器，用于B/S架构的javaweb项目开发
(Browser/Server浏览器/服务器)
jsp,servlet,web框架
